## 主要功能
為學校以系所角度提供服務,以通訊軟體LINE提供的平台為媒介,快捷的架設好系所帳號功能,讓系所能方便創建LINE官方帳號來管理,再以聊天機器人的形式和網站輔助提供服務,以利於增進師生和系所間的交流。
